# HR模拟器 - 操作指南

## 环境要求
- Python 3（macOS 自带）
- 有效的 **DeepSeek API Key**

## 如何运行

### 1. 配置 API Key
将你的 DeepSeek API Key 写入项目根目录的 `apikey.txt`（文件内只放一行 key，无其他内容）：
```
sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### 2. 启动服务器
```bash
cd /Users/a123/HR模拟器
python3 server.py
```
`server.py` 同时提供静态文件服务和 DeepSeek API 代理（`/api/chat`），浏览器不直接调外部API，没有CORS问题。

### 3. 打开游戏
浏览器访问：http://localhost:8000

## 如何添加新数据

### 添加姓名
编辑 `data/names.js`，在数组末尾添加字符串：
```js
const NAMES = [
  // ...已有姓名
  '新名字',
];
```

### 添加技能
编辑 `data/skills.js`，格式同上。

### 添加工作经历
编辑 `data/experiences.js`，添加对象：
```js
{ company: '公司名', role: '职位', years: 3, desc: '一句话描述' }
```

### 添加学历
编辑 `data/educations.js`，格式：`'学校 - 专业'`

### 添加性格特点
编辑 `data/personalities.js`，添加字符串。

### 添加面试问题和回答
1. 在 `data/interviewQuestions.js` 添加：`{ id: 'q7', text: '问题内容' }`
2. 在 `data/interviewAnswers.js` 添加对应回答：
```js
q7: [
  { text: '优秀回答', quality: 'good' },
  { text: '一般回答', quality: 'mid' },
  { text: '糟糕回答', quality: 'bad' },
],
```

### 添加头像
1. 创建 SVG 文件放入 `avatars/` 目录
2. 在 `avatars/avatarList.js` 数组中注册路径

## 补充卡池工具
打开 `tools/补充卡池.html` 可使用 AI 辅助批量生成新数据。

## 文件结构速查
| 目录 | 用途 |
|------|------|
| `css/` | 样式文件，按页面拆分 |
| `js/` | 逻辑文件，按功能拆分 |
| `data/` | 数据池，每种数据独立文件 |
| `avatars/` | SVG头像 + 注册表 |
| `tools/` | 辅助工具 |
